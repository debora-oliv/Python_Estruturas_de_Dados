# Programação Python: Estruturas de Dados

Este repositório contém uma coleção de desafios práticos e algoritmos desenvolvidos para consolidar o domínio sobre as estruturas de dados nativas do Python (Listas, Tuplas, Dicionários e Sets).

O foco foi aprender mais sobre os principais métodos e as principais caraterísticas de cada estrutura, legibilidade e, em alguns casos, otimização de algoritmos.

## 🗂️ Estrutura do Projeto

* **`desafios_listas/`**: Manipulação de matrizes, slicing, algoritmo de janela deslizante (Sliding Window) e Run-Length Encoding.
* **`desafios_tuplas/`**: Imutabilidade, desempacotamento e roteamento espacial (Distância Euclidiana).
* **`desafios_dicionarios/`**: Hashing, complexidade $O(1)$, inversão de chaves e Dictionary Comprehensions.
* **`desafios_conjuntos/`**: Teoria dos conjuntos (união, diferença) e desduplicação.
* **`desafio_agenda/`**: Aplicação dos conceitos em um mini-projeto de **Agenda de Contatos** baseada em operações CRUD na memória.
